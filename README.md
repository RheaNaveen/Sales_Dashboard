# Sales Performance Dashboard

## Project Overview

This project presents a **Sales Performance Dashboard built in Microsoft Excel** to analyze and visualize the performance of sales executives across different regions. The dashboard provides a structured view of sales performance, target achievement, and areas that may require improvement.

While designing the dashboard, I used a **monochromatic blue color theme** to maintain a clean and professional look while keeping the visualizations easy to interpret.

---

## Key Insights from the Dashboard

The dashboard highlights the following performance metrics:

* **Top 5 Sales Executives by Total Sales**
* **Bottom 5 Sales Executives by Total Sales**
* **Top 5 Executives by Target Hit Percentage**
* **Bottom 5 Executives based on Away From Target Percentage**

These insights help quickly identify top performers, evaluate sales contributions, and recognize executives who may require additional support to achieve their targets.
